# Операционные системы
## ДЗ "Запуск своих контейнеров через docker-compose"

## Студент: Козакевич Максим

### Это репа с выполненным заданием 2:

Написать клиент-серверное приложение, которое по запросу метода POST на route "/create" создает пустой текстовый файл. Клиент и сервер должны быть собраны через docker compose. При этом все созданные файлы должны сохраняться через volume.

### Из чего состоит:
1. Клиент. Приложение на Angular, статика отдаётся с помощью nginx. Доступен по localhost:4200. Рендерит кнопку, по нажатию на которую отправляется запрос POST http://localhost:3000/create
2. Сервер. Приложение на NestJS. Доступен по localhost:3000. Создаёт пустой тестовый файл по запросу POST /create, в ответе возвращает имя созданного файла для отображения пользователю

### Как запустить:
1. Клонируем репу через git clone
2. Из корня репы выполняем команду `docker compose build && docker compose up`
3. Переходим на http://localhost:4200/
4. Нажимаем на кнопку "Создать файл". Отобразится имя созданного файла
5. Находим этот файл в volume `os-docker-hw_appdata`
